box-model is a program for simulating physical and chemical processes affecting aerosol inside an aerosol chamber.  It is distributed under license - see COPYING.txt

Authors: Simon O'Meara and David Topping: simon.omeara@manchester.ac.uk

full_move.py for full-moving size structure treatment of size bin population evolution
full_stat.py is the full-stationary size structure
full_move.py is the full-moving size structure
quasi_stat.py is the quasistationary size structure 
hybr_stru.py is the hybrid size structure
